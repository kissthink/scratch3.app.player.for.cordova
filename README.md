Scratch3 App Player
now creating.

# RUN
```
npm install -g cordova
cordova create MyApp
cd MyApp cordova platform add browser
cp -r {this www} ./www
cordova run browser
```

# FOR ANDROID
```
cordova platform add android
cordova build android
cordova run android
```
